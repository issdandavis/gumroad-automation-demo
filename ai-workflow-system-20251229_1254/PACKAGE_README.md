## üõ†Ô∏è Quick Start Guide

### Prerequisites
- Python 3.8+ with pip
- Node.js 18+ with npm (for AI Workflow Architect)
- Git (optional, for updates)

### 1. Extract the Package
```bash
# Extract the ZIP file to your desired location
unzip ai-workflow-system-20251229_1254.zip
cd ai-workflow-system-20251229_1254
```

### 2. Install Python Dependencies
```bash
# Install all required Python packages
pip install -r requirements.txt
```

### 3. Start the Self-Evolving AI System
```bash
cd app-productizer

# Start the main AI system
python evolving_ai_main.py

# Or start the web interface (in a new terminal)
python web_interface.py
```

### 4. Optional: Start AI Workflow Architect
```bash
cd projects_review/AI-Workflow-Architect

# Install Node.js dependencies
npm install

# Start development server
npm run dev
```
