## üì¶ Installation & Setup

### Quick Start (5 minutes)
```bash
# Clone the repository
git clone https://github.com/your-org/self-evolving-ai-framework.git
cd self-evolving-ai-framework

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your API keys

# Initialize the system
python evolving_ai_main.py init

# Run demo
python evolving_ai_main.py demo
```

### Production Deployment
```bash
# Deploy to AWS
python deploy.py --environment production --region us-east-1

# Deploy with Docker
docker build -t evolving-ai .
docker run -d --name evolving-ai-prod evolving-ai

# Deploy to Kubernetes
kubectl apply -f k8s/
```

## üîß API Reference

### Core Framework API
```python
from self_evolving_core import EvolvingAIFramework

# Initialize framework
framework = EvolvingAIFramework()
framework.initialize()

# Get system status
status = framework.get_status()
print(f"Generation: {status['generation']}")
print(f"Fitness: {status['fitness_score']}")

# Propose mutation
mutation = Mutation(
    type="intelligence_upgrade",
    description="Improve response accuracy",
    fitness_impact=5.0
)
result = framework.propose_mutation(mutation)

# Monitor fitness
fitness = framework.get_fitness()
print(f"Success Rate: {fitness.success_rate}")
print(f"Cost Efficiency: {fitness.cost_efficiency}")
```

### REST API Endpoints
```bash
# System status
GET /api/v1/status

# Fitness metrics
GET /api/v1/fitness

# Propose mutation
POST /api/v1/mutations
{
  "type": "protocol_improvement",
  "description": "Optimize API response time",
  "fitness_impact": 3.0
}

# Rollback system
POST /api/v1/rollback
{
```
